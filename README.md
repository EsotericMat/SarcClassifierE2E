# SarcClassifier
End-to-End Sarcasm Detection for social media Comments.

### Overview
SarcClassifier is a two-stage NLP system designed to detect sarcasm in social media comments and classify its type.
The project is end-to-end, covering every step â€” from raw data ingestion and preprocessing, through model training and evaluation, to deployment as a real-time inference service.

### Main Components
1. Data Acquisition
   - Source: [Sarcasm Corpus v2](https://www.kaggle.com/datasets/coldn00ldes/sarcasm-corpus-v2oraby-et-al/data). Contains thousands of labeled Reddit comments annotated for sarcasm. 
   - Split into training, validation, and test sets.
2. Data Preprocessing
   - Cleaning: URLs, emojis, special characters 
   - Normalization: lowercasing, punctuation handling 
   - Tokenization & Lemmatization
3. Feature Engineering
4. Modeling
5. Evaluation
